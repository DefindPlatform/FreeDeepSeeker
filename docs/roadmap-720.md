# FreeDeepSeeker: roadmap на 720 задач

Этот backlog превращает проект из работающего прототипа в поддерживаемый coding-agent продукт. Пункты сгруппированы по подсистемам; каждый пункт должен завершаться кодом или документированным решением, проверкой и записью в changelog при пользовательском влиянии.

## Правила выполнения

- Выполнять небольшими вертикальными срезами; один коммит должен иметь одну понятную цель.
- Не закрывать пункт без автоматической проверки либо сохранённого ручного QA evidence.
- Безопасность, совместимость Windows/Linux и отсутствие утечек секретов являются частью Definition of Done.
- TypeScript-миграция и мультипровайдерная архитектура намеренно не включены согласно выбранному направлению проекта.
